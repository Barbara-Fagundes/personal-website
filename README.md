# Bárbara Fagundes Academic Profile

Static website for an academic profile, designed to be hosted on GitHub Pages.

## Overview

This repository contains an academic profile site with English and Portuguese versions of the main pages. It includes a homepage, biography, industry-facing profile, instructional design portfolio, personal narrative, and contact page.

## Files

- `index.html` — homepage (English)
- `index-pt.html` — homepage (Português)
- `about.html` — about page (English)
- `about-pt.html` — about page (Português)
- `industry.html` — industry-facing profile page (English)
- `industry-pt.html` — industry-facing profile page (Português)
- `instructional-design.html` — instructional design portfolio (English)
- `instructional-design-pt.html` — instructional design portfolio (Português)
- `beyond-cv.html` — personal narrative beyond the CV (English)
- `beyond-cv-pt.html` — personal narrative beyond the CV (Português)
- `contact.html` — contact page (English)
- `contact-pt.html` — contact page (Português)
- `publications.html` — publications page
- `assets/` — images, styles, and static assets
- `files/` — downloadable documents such as the CV

## Deployment

To publish this site on GitHub Pages:

1. Create a repository named `YOUR-USERNAME.github.io`.
2. Upload all files and folders from this repository to the repository root.
3. Add a profile image at `assets/profile.jpg` if desired.
4. Commit and push the repository.
5. Visit `https://YOUR-USERNAME.github.io`.

## Notes

- The site is static HTML and does not require a build step.
- The Portuguese pages are provided alongside the English pages for bilingual support.
- Replace `YOUR-USERNAME` with your GitHub username in the deployment URL.
